If You Know Just a Bit of HTML,
  1. "Clone" the GitHub repository available HERE .
  2. Go to the folder "htmls" and Choose the HTML file that has the name that corresponds to the type of the quantum simulator you want to contribute to (e.g. hybrid).
  3. Make your modification to the information in that HTML file (e.g. correct typos, add new simulators, update informations, etc.).
  4. Once you finish, create a "pull" request on the same GitHub repository HERE .
  5. That's it, you're done! Your contributions will be reviewed as soon as possible to include them in the repository.
  6. Thanks in advance for your contributions!
